## 🔹 Arrays in Java (Theory)

An **array** is a container object that holds a fixed number of values of the same data type. It allows storing multiple values in a single variable instead of creating separate variables.

---

### 🔸 Declaration

Arrays can be declared in two ways:
- dataType[] arrayName  
- dataType arrayName[]  

---

### 🔸 Initialization

Arrays can be initialized in two ways:
- By specifying size (elements get default values)  
- By directly assigning values  

---

### 🔸 Accessing Elements

- Array elements are accessed using an **index**
- Index starts from **0 (zero-based indexing)**  
- Each element can be accessed using its position  

---

### 🔸 length Property

- Every array has a **length property**  
- It returns the total number of elements in the array  

---

### 🔸 Multidimensional Arrays

- Arrays can have multiple dimensions  
- A multidimensional array is an **array of arrays**  
- Commonly used to represent **matrices or tables**  

---

### 🔸 Limitations

- Arrays have a **fixed size** once created  
- Cannot grow or shrink dynamically  
- Can store only **same type of elements**
